# Gaia Backend

## Kurulum

```bash
cd backend
npm install
copy .env.example .env
npm run dev
```

- Frontend varsayılan olarak `http://localhost:8080` üzerinden çalışır.
- Backend varsayılan olarak `http://localhost:3001` üzerinden çalışır.

## Notlar

- `.env` dosyasını git'e eklemeyin.
- `DATABASE_URL` SQLite dosyasının yolunu belirtir.
